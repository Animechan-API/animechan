# `Quote` object

The is the  JSON structure that gets returned for all `/quote` endpoints.

```json
{
	"status": "success",
	"data": {
		"content": "Whenever I counted on someone, I ended up getting hurt.",
		"anime": {
			"id": 2,
			"name": "Hanasaku Iroha"
		},
		"character": {
			"id": 5,
			"name": "Ohana Matsumae"
		}
	}
}
```
