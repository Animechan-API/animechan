# Get a random quote via Anime

This endpoint returns random quote from a given Anime name.

### The endpoint:
```sh
/api/v1/quotes/random?anime=<anime_name>
```

### Return
It will return a [`Quote`](/quote) object

### Example:
> We are using the built-in global `fetch` API here, which is both available in browser and NodeJS as of now. But you can use any other programming language or request lib.

```js
const response = await fetch(`/api/v1/quotes/random?anime=${animeName}`)
const quote = await response.json()
```
